# EduSchedule Pro

Système Intégré de Gestion de l'Emploi du Temps et de Suivi Pédagogique des Séances de Cours.

## Technologies
- Frontend : React 18, Bootstrap 5
- Backend : PHP 8, API REST
- Base de données : MySQL 8 (MariaDB)

## Installation

### Prérequis
- XAMPP (Apache + MySQL)
- Node.js

### Backend
1. Copier le dossier `eduschedule-pro` dans `C:\xampp\htdocs\`
2. Importer `database/eduschedule_pro.sql` dans phpMyAdmin
3. Démarrer Apache et MySQL dans XAMPP

### Frontend
```bash
cd frontend
npm install
npm start
```
